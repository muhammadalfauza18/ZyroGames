# 📖 Tentang Proyek

**ZyroGames** merupakan website penjualan game yang dirancang dengan tampilan modern, responsif, dan interaktif. Website ini dibuat sebagai media untuk menampilkan berbagai pilihan game populer lengkap dengan informasi produk, kategori, serta desain antarmuka yang menarik sehingga memberikan pengalaman pengguna yang lebih baik.

Proyek ini dikembangkan sebagai implementasi pembelajaran **Front-End Web Development**, dengan fokus pada pembuatan antarmuka yang bersih (clean UI), responsif di berbagai perangkat, dan mudah digunakan.

---

# 🚀 Cara Menjalankan Proyek

1. Clone repository

```bash
git clone https://github.com/muhammadalfauza18/ZyroGames.git
```

2. Masuk ke folder project

```bash
cd ZyroGames
```

3. Buka file **index.html**

atau jalankan menggunakan **Live Server** pada Visual Studio Code.
